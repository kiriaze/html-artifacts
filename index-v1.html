How it works

Three steps, no toolchain

Clone, invoke, open. No npm, no build pipeline, no server required.

Step 1
Install
Clone and symlink to ~/.claude/skills/ — takes about 30 seconds.
Step 2
Invoke the skill
Ask Claude to generate any of the 18 patterns for your project context.
Step 3
Open in browser
Fully self-contained HTML — no server, no build, no dependencies.

Install

Get started in 30 seconds

Clone the repo and symlink it into your Claude skills directory. That's it. 

# Clone the skill
git clone https://github.com/kiriaze/html-artifacts \
  ~/.agents/skills/html-artifacts

# Link into Claude skills
ln -s ~/.agents/skills/html-artifacts \
  ~/.claude/skills/html-artifacts

Then ask Claude: Use the html-artifacts skill to generate a product-brief for [your project].

Built on "The Unreasonable Effectiveness of HTML" by Thariq Shihipar (Anthropic). The thesis, categories, and several example shapes are his — the skill scaffolding and recognition heuristic are extensions.
